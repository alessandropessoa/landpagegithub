Minha jornada começou em 2008, ainda no ensino médio técnico do SESI, onde me apaixonei por redes e manutenção de computadores — o que rapidamente me levou a atender pequenas empresas e construir uma base sólida em tecnologia. Em 2009, ingressei no curso técnico em Automação e Controle Industrial no SENAI-CETIND (Bahia) e mergulhei no chão de fábrica: programação de CLPs, instrumentação e projetos para a indústria de petróleo (PetroReconcavo, View Engenharia) e energia (Alstom Power).

Em 2013, fui aprovado em concurso público para o programa de submarinos nucleares da Marinha do Brasil, onde atuei como especialista em automação. A necessidade de integrar dispositivos industriais a sistemas web me levou a aprofundar conhecimentos em engenharia de software, desenvolvendo soluções IoT completas (TI/TO). Em 2019, percebi que o verdadeiro diferencial competitivo estava nos dados: migrei definitivamente para Ciência de Dados, cursei graduação e MBA na área, e atualmente finalizo o Mestrado em Computação na UFF, com foco em Modelagem Neural e Processamento de Linguagem Natural.

Hoje, atuo como Cientista de Dados no CASNAV, contribuindo para o projeto DELFOS — um Sistema de Apoio à Decisão estratégico para o Ministério da Defesa. Minha rotina envolve arquitetar pipelines de dados, desenvolver modelos de IA explicáveis e criar ferramentas de simulação que apoiam decisões de alto impacto, como o Simulador de Paraquedas apresentado no Rio Innovation Week 2025.

Acredito que a verdadeira maturidade analítica está na interseção entre rigor científico, boas práticas de engenharia (MLOps, Clean Architecture) e visão de negócio. Meu propósito é construir pontes entre planilhas estáticas e sistemas inteligentes de tomada de decisão.

🧠 CoDe2CGA – Engenharia de Software Assistida por LLM

LLM (GPT/Gemini) NetworkX Flet AST

Ferramenta de análise estática de código que utiliza Large Language Models para gerar grafos de chamadas (call graphs) a partir do código fonte, extraindo automaticamente a estrutura de classes, métodos e dependências. O sistema calcula métricas de complexidade ciclomática, identifica gargalos de desempenho e sugere refatorações com base em padrões de design.

Diferenciais:

  • Conversa com o código: possibilidade de interagir via chat para esclarecer dúvidas sobre trechos específicos.
  • Seleção granular: análise de arquivos ou diretórios específicos, com opção de ignorar padrões.
  • Análise de tempo: medição do tempo de análise por arquivo, útil para identificar componentes complexos.
  • Troca de modelos e prompts: flexibilidade para experimentar diferentes LLMs e ajustar a estratégia de análise.

Frontend multiplataforma desenvolvido com Flet (Flutter em Python), garantindo execução nativa em desktop, web e mobile.

📈 Previsão de Demanda para Implementos Rodoviários

Prophet ARIMA/SARIMAX Random Forest NLP (BERT) Streamlit

Sistema completo de forecasting desenvolvido para uma consultoria do setor de implementos rodoviários. A solução combina modelos estatísticos clássicos (ARIMA, SARIMAX) com machine learning (Random Forest) para prever preços e volumes de vendas, incorporando variáveis exógenas como taxa SELIC e indicadores macroeconômicos.

O grande diferencial está na criação de um índice de sentimento de mercado derivado da análise de notícias setoriais (coletadas via web scraping). Esse índice é usado para gerar um KPI proprietário chamado "Temperatura de Mercado", que quantifica a pressão altista/baixista sobre os preços. O sistema também inclui um módulo de storytelling dinâmico, que gera narrativas automáticas em linguagem natural para contextualizar as previsões.

Arquitetura conteinerizada com Docker, pipelines de dados versionados com DVC e dashboard interativo em Streamlit.

🏗️ ERP com Orquestração Multiagente para Obras

Django Docker Sistemas Multiagente Gantt/Curva S

Sistema de gestão empresarial verticalizado para construtoras, com módulos de cronograma físico-financeiro, quadro Kanban, gráfico de Gantt e curva S. O backend em Django expõe uma API REST consumida por um sistema multiagente que auxilia na elaboração automática de cronogramas, otimizando a alocação de recursos e a mitigação de riscos de atraso.

O projeto segue os princípios de Clean Architecture e utiliza Docker para orquestração dos serviços, garantindo escalabilidade e facilidade de implantação.

🔒 Machine Unlearning para Conformidade com a LGPD

BERT Random Forest SISA LGPD

Implementação do framework SISA (Sharded, Isolated, Sliced, Aggregated) para viabilizar o "direito ao esquecimento" em modelos de machine learning. Utilizando embeddings do BERT como features de entrada, treinamos múltiplos modelos Random Forest em partições (shards) independentes dos dados. Quando um titular solicita a exclusão de seus dados, apenas os shards afetados precisam ser retreinados, reduzindo drasticamente o custo computacional e garantindo conformidade legal.
